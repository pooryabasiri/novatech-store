# NovaTech Store — Modern E-Commerce Frontend
  A fully-featured e-commerce frontend built entirely with Vanilla JavaScript, HTML5, and CSS3 — without using any framework.

##  Live Demo
  https://novatech-store-neon.vercel.app/

##  Overview
  NovaTech Store is a modern, production-grade e-commerce frontend designed to demonstrate that a complete shopping experience can be built without relying on heavy frameworks like React or Vue.
  The project focuses on clean modular architecture, modern JavaScript patterns, accessibility, and responsive design — all delivered through pure Vanilla JavaScript and ES Modules.
  The only external library used is Swiper.js for sliders.

---

##  Features

  - Full product catalog with filtering, sorting, and live search
  - Product details page with image gallery and quantity selector
  - Related products and recently viewed history
  - Cart sidebar with real-time updates
  - Dedicated cart page with coupons, tax, and shipping calculation
  - 3-step checkout flow (Shipping → Payment → Confirmation)
  - Order success page with printable receipt
  - Wishlist sidebar accessible from any page
  - Dark mode with persistent preference
  - Fly-to-cart animation
  - Toast notifications
  - Skeleton loaders
  - Scroll reveal animations
  - Hero and featured product sliders
  - Fully responsive mobile-first design
  - Keyboard accessible and ARIA-labeled
  - LocalStorage persistence for cart, wishlist, and theme

---

## 🛠️ Tech Stack

  - HTML5
  - CSS3 (Custom Properties, Grid, Flexbox)
  - Vanilla JavaScript (ES6+ Modules)
  - Swiper.js
  - LocalStorage API
  - Fetch API

This project was built to demonstrate:

  - Strong command of core JavaScript fundamentals
  - Building scalable applications without frameworks

  - Event delegation and performance optimization
  - Responsive and accessible UI development
  - Real-world e-commerce UX patterns
